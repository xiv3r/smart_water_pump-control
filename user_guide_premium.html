01. Operational Architecture

The APC-S3 Pro utilizes a True Dual-Core RTOS architecture. Core 1 is isolated for real-time safety critical tasks (Sensor Polling, Motor Control, Solenoid Venting), while Core 0 handles the Network Stack (WiFi, MQTT, Local Web Server). This ensures the pump never fails even during heavy network congestion.

🏠 စနစ်၏ တည်ဆောက်ပုံ - ဤစက်တွင် ပရိုဆက်ဆာ နှစ်ခု (Dual-Core) သီးခြားစီ တည်ရှိပါသည်။ ပထမပရိုဆက်ဆာသည် မော်တာ၏ ဘေးကင်းလုံခြုံရေးနှင့် ဆင်ဆာများကိုသာ အချိန်ပြည့်စောင့်ကြည့်နေပြီး ဒုတိယ ပရိုဆက်ဆာမှာ အင်တာနက်နှင့် ချိတ်ဆက်မှုကို တာဝန်ယူပါသည်။ ထို့ကြောင့် အင်တာနက် လိုင်းနှေးနေသော်လည်း မော်တာသည် ဘေးကင်းစွာ ဆက်လက် အလုပ်လုပ်နေမည် ဖြစ်ပါသည်။

06. 💧 Precision Water Level Logic

The system polls the Ultrasonic sensor every 4 seconds and uses a 20-sample Median Filter to ignore water turbulence. The Start/Stop thresholds are now fully configurable (Low: 20%-70%, Full: 80%-100%) via the cloud or local dashboard to match your specific tank geometry.

💧 ရေမှတ်ထိန်းချုပ်မှုစနစ် - ရေတိုင်ကီ၏ အတက်အကျကို ၄ စက္ကန့်လျှင် တစ်ကြိမ် တိုင်းတာပါသည်။ ယခုအခါ ရေစတင်တင်မည့် ရေမှတ် (20%-70%) နှင့် ရေအပြည့်သတ်မှတ်ချက် (80%-100%) တို့ကို မိမိတိုင်ကီအမျိုးအစားအလိုက် စိတ်ကြိုက် ချိန်ညှိနိုင်ပြီ ဖြစ်ပါသည်။

07. 🌙 Smart Scheduling (DND)

To prevent motor noise during sleeping hours, the system includes a Do Not Disturb (DND) timer. When active, a Moon Badge appears on the dashboard, and the pump will not auto-start even if the water is low.
Note: You can still force-start the pump via the Manual Button or Cloud during DND.

🌙 ညဘက် အသံပိတ်စနစ် (DND) - ညအိပ်ချိန်တွင် မော်တာသံကြောင့် အနှောင့်အယှက်မဖြစ်စေရန် မိမိအိပ်ရာဝင်ချိန်နှင့် နိုးမည့်အချိန်ကို သတ်မှတ်ထားနိုင်ပါသည်။ ၎င်းအချိန်အတွင်း ရေနည်းနေသော်လည်း အလိုအလျောက် ရေတင်ပေးမည် မဟုတ်ပါ။ (အရေးပေါ် လိုအပ်ပါက Manual ခလုတ် သို့မဟုတ် ဖုန်းမှတဆင့် နှိပ်၍ တင်နိုင်ပါသည်)။

08. ⚡ Voltage Security Engine

The system constantly monitors the AC mains voltage via the ZMPT sensor. To protect the motor from burnout, the system now features Customizable Power Guards.

⚠️ Cut-off Limits

Configurable: Low 150V-190V | High 230V-260V. Precise protection for your motor windings.

⚖️ Resume Gap

Configurable 1V to 10V. Prevents rapid on/off switching (chattering) when voltage is hovering near the threshold.

⚡ ဗို့အား ကာကွယ်ခြင်း - မီးအား အတက်အကျကြောင့် မော်တာလောင်ခြင်းမှ ကာကွယ်ရန် ဗို့အားမြင့်/နိမ့် သတ်မှတ်ချက်များကို စိတ်ကြိုက်ပြင်ဆင်နိုင်ပါသည်။ မီးအားပြန်တက်လာချိန်တွင် စက်ခဏခဏ ပိတ်ဖွင့်မဖြစ်စေရန် Resume Gap (1V-10V) စနစ်ကိုလည်း ထည့်သွင်းပေးထားပါသည်။

09. 🌊 Dry-Run & Auto-Retry

1. Configurable Dry-Run Safety

When the motor starts, the system monitors water flow. You can now adjust the detection sensitivity and retry behavior.

⏱️ Dry-Run Delay

Configurable: 60s to 180s. Set the timeout duration before the "DRY ALRM" is triggered.

🔄 Auto-Retry Time

Options: OFF, 30m, 60m. Automatically attempts to restart the pump to check if water has returned.

🌊 ရေပြတ်လပ်မှု ကာကွယ်ခြင်း - ရေပြတ်လပ်မှု စောင့်ဆိုင်းချိန် (စက္ကန့် ၆၀ မှ ၁၈၀) နှင့် အလိုအလျောက် ပြန်လည်စမ်းသပ်မည့်အချိန် (မိနစ် ၃၀ သို့မဟုတ် ၆၀) တို့ကို စိတ်ကြိုက် ရွေးချယ်နိုင်ပါသည်။

Automatic Retry & Alarm Sequence

⚠️
Two-Stage Dry-Run Protection Cycle:
  1. Stage 1: Active Alarm (DRY ALRM - 60 seconds): If the flow sensor detects no water for the configured Dry-Run Delay, the pump stops immediately. The LED flashes red, the LCD displays DRY ALRM, and the buzzer beeps urgently (500ms intervals) for exactly 60 seconds.
  2. Stage 2: Lockout / Waiting Retry (LOCKED / WAITING_RETRY!): After 60 seconds, the buzzer stops beep patterns.
    • If Auto-Retry is enabled (e.g., 30 or 60 minutes): The LCD shows LOCKED and status reads WAITING_RETRY!. A countdown will run on the screen. Once the minutes expire, the system automatically clears the error and restarts the pump to test for water.
    • If Auto-Retry is set to OFF: The system enters permanent lockout (PUMP_LOCKED!). The motor will not restart automatically and must be reset manually.

How to Reset Manually: Press the physical button once or send a Reset command via the MQTT Cloud API or Local Web UI to instantly clear the dry-run error and restart operation.

🔄 ရေပြတ်လပ်မှု အလိုအလျောက် ပြန်လည်စမ်းသပ်ခြင်းနှင့် အချက်ပေးစနစ် -

၁။ အဆင့် ၁ - အချက်ပေးအဆင့် (DRY ALRM - ၆၀ စက္ကန့်): ရေပြတ်လပ်သွားလျှင် မော်တာချက်ချင်းရပ်သွားပြီး LED အနီရောင်ဖလက်ရှ် လင်းပါမည်။ LCD မျက်နှာပြင်တွင် DRY ALRM ဟုပြသပြီး ဘတ်ဇာမှ ၆၀ စက္ကန့်တိတိ အသံလှိုင်းတိုဖြင့် အချက်ပေးမြည်ပါမည်။
၂။ အဆင့် ၂ - စောင့်ဆိုင်း/ပိတ်ဆို့အဆင့် (LOCKED / WAITING_RETRY!): ၆၀ စက္ကန့်ပြည့်ပါက ဘတ်ဇာရပ်သွားပါမည်။
  • Auto-Retry ဖွင့်ထားပါက (မိနစ် ၃၀ သို့မဟုတ် ၆၀): LCD တွင် LOCKED ဟုပြပြီး ပြန်လည်စမ်းသပ်ရန် စက္ကန့်/မိနစ် Countdown ပြေးနေပါမည်။ အချိန်စေ့ပါက မော်တာကို အလိုအလျောက် ပြန်လည်မောင်းနှင် စမ်းသပ်ပါမည်။
  • Auto-Retry ပိတ်ထားပါက (OFF): စက်သည် လုံးဝရပ်တန့်နေမည်ဖြစ်ပြီး (PUMP_LOCKED!) အလိုအလျောက် ပြန်မလည်ပါ။
လက်ဖြင့် ပြန်လည်စတင်ပုံ (Reset): စက်ရှိ Manual ခလုတ်ကို တစ်ချက်နှိပ်ခြင်းဖြင့်သော်လည်းကောင်း၊ Web Dashboard သို့မဟုတ် MQTT Cloud မှ Reset ပေးခြင်းဖြင့်သော်လည်းကောင်း အမှားကို ချက်ချင်းဖြေရှင်းပြီး မော်တာကို ပြန်လည်ပတ်စေနိုင်ပါသည်။

2. Manual Button Control (Industrial Logic)

The physical button on GPIO 9 provides a robust multi-state interface for local control.

1. Single Press: Toggles the motor ON or OFF manually. The system waits 850ms after release to confirm it is a single-click (not the start of a double-click). If a block condition exists, the LED flashes red and the buzzer beeps once.

2. Double Click: On the second press downstroke (within 850ms of the first release), the system instantly displays the System Info screen on the LCD for 10 seconds — showing the Device IP Address, Cloud ID, and Remaining License Days. Two short beeps confirm the action.

3. Long Press (3 seconds): Forces the device to enter AP Configuration Mode (SSID: Auto-Pump-Config / Password: 12345678). A 1-second buzzer confirms activation. If no client connects within 5 minutes, the device automatically exits AP mode.

🎮 Manual ခလုတ် အသုံးပြုပုံ - GPIO 9 ရှိ ခလုတ်ကို အောက်ပါအတိုင်း အသုံးပြုနိုင်ပါသည်။

၁။ တစ်ချက်နှိပ်လျှင်: မော်တာကို ဖွင့်ခြင်း/ပိတ်ခြင်း ပြုလုပ်ပါမည်။ (850ms ခေတ္တစောင့်သည်) ပိတ်ဆို့မှုရှိပါက LED အနီရောင် လက်ဒ် လင်းပြမည်ဖြစ်သည်။
၂။ နှစ်ချက်ဆင့်နှိပ်လျှင်: ပထမနှိပ်ပြီး 850ms အတွင်း ဒုတိယ ဆင်းသက်မြောက်သောအခါ LCD မျက်နှာပြင်တွင် 10 စက္ကန့် ကြာ System Info (IP၊ ID နှင့် License ရက်) ပြသပေးပါမည်။ တိုင်ထောင်ရှင် ၂ ချက် မြည်ပါမည်။
၃။ ၃ စက္ကန့် ဖိထားလျှင်: ဝိုင်ဖိုင်အသစ် ပြန်လည်ချိန်ညှိရန် AP Mode (Auto-Pump-Config) သို့ ဝင်ရောက်ပါမည်။ ဘတ်ဇာ ၁ စက္ကန့် မြည်ပါမည်။ ၅ မိနစ်အတွင်း မည်သူမျှ မချိတ်ဆက်ပါက AP Mode မှ အလိုအလျောက် ပြန်ထွက်သွားပါမည်။)

10. ❄️ Thermal Cool-Down Protection

2. True Offline Fallback (No Internet Needed)

The APC-S3 Pro is built for **mission-critical reliability**. If your home router loses internet connection, the system works independently using your previously saved configuration. It will continue all automatic pumping, safety checks, and scheduling tasks without interruption. Once your internet is restored, the system seamlessly reconnects to the Cloud and synchronizes all data automatically without any manual intervention or even being noticed. For local dashboard access during an outage, simply press and hold the manual button for 3 seconds to broadcast the Auto-Pump-Config network.

3. Sync Heartbeat Fallback (5m): In a Master/Slave configuration, if the wireless "Heartbeat" is lost for more than 5 minutes, the Slave unit will automatically revert to Standalone Mode to ensure water availability isn't blocked by a network failure.

📶 အင်တာနက်ပြတ်တောက်သွားသည့် အခြေအနေ (Autonomous Mode) - အကယ်၍ အိမ်ရှိ အင်တာနက်လိုင်း ပြတ်သွားပါက စက်သည် ယခင်က သိမ်းဆည်းထားသော Settings များအတိုင်း မိမိဘာသာ လွတ်လပ်စွာ ဆက်လက် အလုပ်လုပ်နေမည် ဖြစ်သည်။ အင်တာနက်ပြန်ရပါက မည်သည့်အရာမျှ ပြုလုပ်ရန်မလိုဘဲ Cloud စနစ်သို့ အလိုအလျောက် ချောမွေ့စွာ ပြန်လည်ချိတ်ဆက်ပြီး ဒေတာများကို Sync ပြုလုပ်ပေးမည် ဖြစ်ပါသည်။

17. 🔗 Master & Slave Synchronization

The APC-S3 Pro v2.1.0 introduces the Sequential Priority Matrix, allowing a Master unit (at the Sump Tank) to coordinate with a Slave unit (at the Upper Tank) wirelessly via HiveMQ MQTT or ESP-NOW. This prevents the pump from starting if the supply source is insufficient or if another priority tank is being served.

🎛️ Operational Roles

Mode 0: Standalone Device.
Mode 1: Master (Authority / Sump Control).
Mode 2: Slave (Consumer / Requestor).

🛡️ Safety Protocols

Settling Water: Prevents silt intake by waiting for sediment to settle (Purple LED).
Heartbeat: 5-min fallback to local logic if network drops.

Sequential Priority Interlock

🔗
How the Slave decides to start:

The Slave continuously monitors the Master's published info field via MQTT. The Slave is only permitted to run its pump when the Master's reported state is either SYSTEM_STANDBY! or contains FLOW (i.e., FLOW_DETECTED! or FLOW_CHECKING!).

If the Master is Pumping (pStat = ON), Settling, in Voltage Error, Cooling Down, or in any alarm state — the Slave's pump is immediately stopped and blocked until the Master returns to a safe state.

Link Timeout (5 minutes): If no update is received from the Master for 5 minutes (network drop), the Slave automatically falls back to Standalone Mode and operates independently using its own sensor data.

🔗 မာစတာနှင့် ဆလိဖ် ချိတ်ဆက်မှု - ဤစနစ်သစ်တွင် Master (အောက်ကန်) နှင့် Slave (အပေါ်ကန်) တို့ကို ကြိုးသွယ်ရန်မလိုဘဲ MQTT ဖြင့် ချိတ်ဆက်ထားပါသည်။

Slave သည် Master ၏ SYSTEM_STANDBY! သို့မဟုတ် FLOW_DETECTED!/FLOW_CHECKING! အခြေအနေတွင်သာ မော်တာ စတင်ခွင့်ရပါသည်။ Master ၏ Pump ဖွင့်နေစဉ်၊ SETTLING နေစဉ် သို့မဟုတ် Error အခြေအနေတွင် Slave မော်တာ ချက်ချင်းရပ်ပေးမည်ဖြစ်သည်။
Purple LED (ခရမ်းရောင်): Master ၏ Settling Water အဆင့်တွင် ရေအနည်ထိုင်ရန် ကြာမျှ ကျန်ရှိသည်ကို SETTL XXM ဖြင့် LCD တွင် ပြသပါသည်။
Link Timeout (5 မိနစ်): Master မှ ၅ မိနစ် ဆက်သွယ်မှု မရပါက Slave သည် Standalone Mode သို့ အလိုအလျောက် ပြောင်းသွားပါမည်။

19. ❓ Frequently Asked Questions

The pump won't turn on and I can't hear the buzzer. Is it broken?
Not necessarily. Check the screen. It might be in quiet hours (moon icon), the tank might already be full, or the electricity may be unstable. Wait a few minutes and check again.
မော်တာ မလည်ဘဲ အချက်ပေးသံလည်း မကြားရပါက ဘာစစ်ဆေးရမလဲ? - မျက်နှာပြင်ကို အရင်ကြည့်ပါ။ Moon Icon (ညဘက် အသံပိတ်ချိန်) ဖြစ်နေခြင်း၊ ရေပြည့်နေခြင်း သို့မဟုတ် မီးအား မမှန်ခြင်းတို့ ဖြစ်နိုင်ပါသည်။ ခဏစောင့်ပြီး ပြန်ကြည့်ပေးပါ။
Does the system need the internet to function automatically?
No. The system is designed for Autonomous Reliability. If internet access is lost, the device continues to work independently based on your previously saved settings. Once the internet connection resumes, the system will automatically reconnect and sync to the cloud seamlessly without any user intervention.
အင်တာနက်မရှိဘဲ စက်အလုပ်လုပ်ပါသလား? - ဟုတ်ကဲ့၊ အလုပ်လုပ်ပါသည်။ ဤစက်ကို Autonomous Reliability အတွက် အထူးထုတ်လုပ်ထားသောကြောင့် အင်တာနက်ပြတ်တောက်သွားသော်လည်း ယခင်က သတ်မှတ်ထားသော Settings များအတိုင်း မိမိဘာသာ လွတ်လပ်စွာ ဆက်လက်အလုပ်လုပ်နေမည် ဖြစ်ပါသည်။ အင်တာနက်ပြန်ရသည်နှင့် တစ်ပြိုင်နက် မည်သည့်အရာမျှ ပြုလုပ်ရန်မလိုဘဲ Cloud စနစ်သို့ အလိုအလျောက် ပြန်လည်ချိတ်ဆက်သွားမည် ဖြစ်ပါသည်။
The screen says "FLOW CHK" and the pump has been running for a minute. Is that ok?
If it has been on for more than 1–2 minutes and still says FLOW CHK, check that water is reaching the pump. The pipe might be blocked or the water supply might be off.
မျက်နှာပြင်တွင် "FLOW CHK" ပြပြီး မော်တာလည်နေသည်မှာ ၁ မိနစ်ကျော်နေပါက ပုံမှန်လား? - ၂ မိနစ်ကျော်အထိ FLOW CHK ပြနေပါက ရေအမှန်တကယ် တက်မလာခြင်း ဖြစ်နိုင်ပါသည်။ ပိုက်လိုင်းပိတ်နေခြင်း သို့မဟုတ် ရေအရင်းအမြစ်ကုန်နေခြင်း ရှိမရှိ စစ်ဆေးပါ။
How can I change my settings if my home Wi-Fi is down?
Press and hold the Manual Button for 3 seconds to activate the Direct Configuration Hotspot. Connect your phone to "Auto-Pump-Config" and visit 192.168.4.1 to manage settings offline.
ဝိုင်ဖိုင်မရှိဘဲ Settings များကို ဘယ်လိုပြင်မလဲ? - Manual ခလုတ်ကို ၃ စက္ကန့် ဖိနှိပ်၍ Direct Config စနစ်ကို ဖွင့်ပါ။ ထို့နောက် ဖုန်းဖြင့် ၁၉၂.၁၆၈.၄.၁ သို့ တိုက်ရိုက်ဝင်ရောက် ပြင်ဆင်နိုင်ပါသည်။
What is the default 6-Digit PIN for remote access?
The default factory PIN is 123456. For your security, it is highly recommended to change this to a private number via the web dashboard.
စက်၏ ၆ လုံးတွဲ PIN နံပါတ် အစောပိုင်းမှာ ဘယ်လောက်လဲ? - စက်ရုံထုတ် PIN မှာ ၁၂၃၄၅၆ ဖြစ်ပါသည်။ လုံခြုံရေးအတွက် မိမိစိတ်ကြိုက် နံပါတ်အသစ်သို့ ချက်ချင်းပြောင်းလဲရန် အကြံပြုအပ်ပါသည်။
The light turned red and the buzzer is going off. What do I do?
Check the screen — it will say what the problem is (dry alarm or sensor issue). Press the button once to acknowledge and follow troubleshooting steps.
မီးနီလင်းပြီး အချက်ပေးသံမြည်နေပါက ဘာလုပ်ရမလဲ? - မျက်နှာပြင်ကို ကြည့်ပါ။ "dry alarm" သို့မဟုတ် "sensor error" လားဆိုသည်ကို သိရပါမည်။ ခလုတ်ကို တစ်ချက်နှိပ်၍ အချက်ပေးသံကို ပိတ်ပြီးနောက် ပြဿနာကို ဖြေရှင်းပါ။
The screen shows "NO WIFI" but I didn't change anything.
Your home router may have restarted. Wait 2–3 minutes — the device will reconnect automatically. If it stays on NO WIFI for >5 minutes, restart the device.
"NO WIFI" ပြနေပါက ဘာလုပ်ရမလဲ? - ဝိုင်ဖိုင် Router ပိတ်သွားခြင်း သို့မဟုတ် Restart ဖြစ်နေခြင်း ဖြစ်နိုင်သည်။ ၂-၃ မိနစ် စောင့်ပါက စက်မှ အလိုအလျောက် ပြန်ချိတ်ပါလိမ့်မည်။ ၅ မိနစ်ကျော်သည်အထိ မချိတ်ပါက စက်ကို ပိတ်ပြီး ပြန်ဖွင့်ပေးပါ။
Can someone else control my pump from far away?
Only someone who knows your Device PIN can send remote commands. Make sure your PIN is kept private. Your installer set the PIN during setup.
အခြားသူများက ကျွန်ုပ်၏ မော်တာကို အဝေးမှ ထိန်းချုပ်နိုင်ပါသလား? - မရပါ။ သင်၏ ၆ လုံးတွဲ PIN နံပါတ်ကို သိသူသာလျှင် ထိန်းချုပ်နိုင်မည် ဖြစ်ပါသည်။ မိမိ၏ PIN နံပါတ်ကို လျှို့ဝှက်ထားပါ။
The pump is on but no water is coming out of my tap.
The screen likely shows FLOW CHK (still priming) or the pipe has an issue. Check for closed valves between the pump and the tap.
မော်တာလည်နေသော်လည်း ရေမထွက်ပါက ဘာဖြစ်နိုင်သလဲ? - "FLOW CHK" ပြနေပါက ရေစတင်တက်ရန် အချိန်ယူနေခြင်း ဖြစ်နိုင်သည်။ ပိုက်လိုင်းရှိ အဆို့ရှင်များ (Valves) ပိတ်နေခြင်း ရှိမရှိ စစ်ဆေးပါ။
What happens during a power cut?
When power returns, the device restarts automatically. It remembers the last pump setting and will resume normal operation within 30 seconds.
မီးပျက်သွားပါက ဘာဖြစ်မလဲ? - မီးပြန်လာပါက စက်သည် အလိုအလျောက် ပွင့်လာပါမည်။ ယခင်က သိမ်းဆည်းထားသော Settings များအတိုင်း စက္ကန့် ၃၀ အတွင်း ပုံမှန်အတိုင်း ပြန်လည် အလုပ်လုပ်ပါမည်။
